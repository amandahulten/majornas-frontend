# Majornasbocker

## Installation

**Requirements:**

- Node.js version 16.14.0 or higher
- NPM or Yarn

1. Clone this project
```bash
git clone https://github.com/amandahulten/majornas-frontend.git
```
2. Install dependencies
```bash
npm install
```
3. Preview the website locally
```bash
npm run dev
```
